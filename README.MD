&nbsp;Pokémon Info Fetcher



A Python script that retrieves detailed information about any Pokémon using the \[PokeAPI](https://pokeapi.co/) and outputs it to a text file.



\## Features

\- Fetch Pokémon \*\*name, types, stats, and abilities\*\* with descriptions.

\- Calculate \*\*weaknesses and resistances\*\* including ×4, ×2, ×0.5, and ×0 damage multipliers.

\- Outputs results to a \*\*.txt file\*\* and automatically opens it for convenience.

\- Standalone \*\*executable\*\* included for users who don’t have Python installed.



\## Tools \& Libraries Used

\- Python 3.11

\- `requests` for API calls

\- `os` for file handling and opening the output

\- PokeAPI (https://pokeapi.co/)



\## How to Run

1\. Open the `Runnable EXE` folder.  

2\. Double-click `PokemonScript.exe`.  

3\. Enter the Pokémon name or ID.  

4\. The script will create a `.txt` file with all details and automatically open it.  



\## Example Output

An example file for \*\*Pikachu\*\* can be found in the `Example Results` folder.



\## Notes

\- Works with Pokémon names or their National Dex ID. The file is named after what you searched. 

\- The script requires an internet connection to fetch data from PokeAPI.
